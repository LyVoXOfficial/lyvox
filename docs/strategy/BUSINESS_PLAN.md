# LyVoX — Бизнес-план (модель)

Дата снимка: 2026-07-03 · Статус: историческая рабочая модель, основанная на состоянии проекта на эту дату.

> ⚠️ Все денежные оценки — модельные допущения для планирования, не прогноз. Юридические выводы требуют бельгийского counsel (см. F3/F4/F5).

---

## 1. Резюме

**LyVoX** — trust-first C2C/B2C маркетплейс для Бельгии: локальные объявления, верифицированные продавцы, анти-скам-модерация, 5 языков (NL/FR/DE/EN/RU). Технически платформа уже живая (www.lyvox.be): каталоги, поиск с FTS, чат с анти-фрод-скраббингом контактов, trust-score, бизнес-аккаунты с проверкой KBO/VIES, DSA-панель трейдера, отзывы, биллинг бустов на Stripe.

**Ключевая ставка:** у действующих игроков (2dehands/2ememain, Facebook Marketplace) доверие — слабое место. Скам-волна в Бельгии (фишинг «покупателей» со ссылками на фейковые курьерские оплаты) создаёт спрос на площадку, где верификация и безопасная коммуникация — ядро продукта, а не надстройка.

**Модель монетизации двухфазная**, жёстко разделённая юридическим гейтом:
- **Деньги ①** (живёт уже сейчас, без компании): LyVoX продаёт продавцам собственные услуги — бусты объявлений (live), Pro-подписка для бизнеса (построена, за флагом).
- **Деньги ②** (только после регистрации компании + PSD2/AML-гейт F3): escrow/safe-deal, комиссия с транзакций, in-chat оплата Bancontact, интегрированная доставка.

## 2. Рынок

| Параметр | Оценка |
|---|---|
| Население Бельгии | ~11,8 млн |
| Основной конкурент | 2dehands.be / 2ememain.be (Adevinta) — де-факто монополист общих объявлений |
| Прочие | Facebook Marketplace (без защиты), Vinted (мода), Immoweb (недвижимость), AutoScout24 (авто) |
| Недообслуженные сегменты | экспаты/диаспоры (EN/RU/DE-аудитории), покупатели, «обжёгшиеся» на скаме, малый бизнес без входа в e-commerce |

**Позиционирование:** не «ещё одна доска», а «место, где сделка безопасна»: verified-бейджи, trust-score, чат внутри платформы, Car-Pass/EPC-бейджи по бельгийским категорийным реалиям, 5 языков (у 2dehands — NL/FR).

## 3. Продукт и дифференциация (что уже есть в коде)

1. **Trust-ядро:** производный trust-level из таблицы `verifications` (email/phone/itsme-ready), trust-score F14, бейджи, отзывы с чат-гейтом (нельзя накрутить без реального диалога по объявлению).
2. **B2C-комплаенс как фича:** бизнес-онбординг с офлайн-проверкой KBO (MOD-97) + VIES VAT, публичная DSA-панель трейдера. Порог входа для честного бизнеса — минуты, для мошенника — барьер.
3. **Анти-фрод:** disposable-email блок, rate-limit, Turnstile (активация ключами), скраббинг контактов в чате, fraud-engine (F9), модерация AI (edge function).
4. **Категорийная глубина:** каталожные схемы по доменам (транспорт/недвижимость/электроника…), базы знаний поколений авто (pros/cons/inspection tips) — контент, который агрегаторы объявлений не дают.
5. **i18n-паритет 5 локалей** с guard-тестом — конкурентное преимущество на двуязычном рынке + экспаты.

## 4. Модель монетизации

### Фаза A (сейчас, company-free) — «Деньги ①»
| Поток | Статус | Модель |
|---|---|---|
| Бусты объявлений | **LIVE** (Stripe `mode:payment`) | €2–15 за буст/пакет, дифференциация по категории (авто/недвижимость дороже) |
| Pro-подписка (бизнес-кабинет) | Построена, за флагом `CAPABILITY_PRO_SUBSCRIPTIONS` | €15–39/мес: лимиты объявлений ↑, бейдж, статистика (feature 58) |
| Категорийные листинг-пакеты | Roadmap (39) | Пакеты для автодилеров/агентств: N активных объявлений |

### Фаза B (после регистрации компании + F3) — «Деньги ②»
| Поток | Гейт | Модель |
|---|---|---|
| Safe-deal / escrow (10) | ⛔ F3 PSD2/AML + Stripe Connect | 3–5% с покупателя (страховка сделки), как у 2dehands «Kooop veilig» |
| Bancontact in-chat (13) | ⛔ F3 | вкл. в escrow-fee |
| Интегрированная доставка Bpost/PUDO (12) | договор с Bpost | маржа на лейблах €0,5–1,5 |
| Реклама/promoted placement | масштаб ≥100k MAU | CPM/CPC |

### Модельная юнит-экономика Фазы A (допущения)
- Активный продавец платит в среднем €1,2/мес (5–8% продавцов покупают буст ~€8, 1–2% бизнесов — Pro €25).
- CAC органический ≈ €0 (SEO/соцсети/сарафан), платный тест ≈ €3–8/регистрация.
- Инфраструктура: Vercel+Supabase+Upstash ≈ €100–300/мес до ~50k MAU; предельные издержки на пользователя →0.
- **Break-even по инфраструктуре:** ~300–500 платящих действий/мес — достижимо на одной нише (авто) в одном городе.

| Фаза | Условие входа | Цель | Монетизация |
|---|---|---|---|
| **A0 — Soft-launch** (сейчас) | done: security-lockdown, cookie-consent, legal-скелет, erasure | ликвидность в 1–2 нишах, 1000 объявлений, первые 100 WAU | бусты (уже live) |

| **C — Платформа** | 100k+ MAU | вертикали (авто/недвижимость), реклама | + promoted, вертикальные подписки |

## 6. Юридические гейты (блокеры бизнес-модели)

1. **F3 PSD2/AML** — «деньги ②» не строить в прод до письменного вывода counsel + Stripe Connect review (`docs/features/escrow-legal-gate.md`). Детонатор: DAC7-репортинг + VAT deemed-supplier + строгий DSA Art.30.
2. **Ставка DSA Art.30** — пока LyVoX contact-only, трейдер-верификация опирается на consumer law (KBO/VAT без ID-копии). Если counsel скажет, что Art.30 применим и к доске контактов — ID-копия становится Phase-A-зависимостью.
3. **F4 GDPR** — механики (consent, erasure, export) построены; RoPA/DPIA/DPA-реестр — на подпись основателю (6 процессоров).
4. **DAC7** — включается только с «деньгами ②»; в Фазе A отчётности нет, т.к. платформа не посредничает в оплате.

## 7. Риски

| Риск | Вероятность | Митигация |
|---|---|---|
| Cold-start ликвидности (главный) | высокая | нишевой клин + город-за-городом (см. GTM), сид бизнес-продавцами, «пустых полок» не показывать (zero-result → SaveSearch уже есть) |
| Ответ 2dehands (копирование trust-фич) | средняя | скорость + глубина (KB, 5 языков, RU/EN-сегмент им неинтересен) |
| Юр-риск Art.30/PSD2 | средняя | гейты F3/F5, contact-only до counsel |
| Фрод-инцидент, репутация | средняя | анти-фрод-стек уже built-in; инцидент-плейбук в модерации (38) |
| Основатель-узкое-место (активации) | высокая | чек-лист активаций уже сформирован (Turnstile, CRON_SECRET, Stripe Pro price, legal text) |

## 8. KPI-дерево

- **North Star: успешные контакты покупатель↔продавец/нед** (чат-диалоги с ≥3 сообщениями двух сторон) — прокси сделки, пока транзакция off-platform.
- Supply: новые объявления/нед, % с фото, % от verified-продавцов (виден в hero: «верифицированных в выдаче»).
- Demand: поиски/нед, CTR выдача→карточка, saved searches.
- Trust: % объявлений от L1+ продавцов, report-rate, время реакции модерации.
- Revenue: буст-конверсия продавцов, MRR Pro, ARPU платящего.

## 9. Что нужно от основателя сейчас (не код)

1. Регистрация компании (или решение об eenmanszaak/KBO) — открывает Фазу B и itsme.

3. Ключи: Turnstile (free), CRON_SECRET, Stripe recurring Price для Pro.
4. Подписать 6 DPA (Supabase/Stripe/Twilio/Cloudflare/Upstash/Vercel).
5. Выбрать клин-нишу и город запуска (рекомендация — в GTM-плане).
