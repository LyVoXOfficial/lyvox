## 1. Стратегическая рамка

Маркетплейс умирает от пустоты, а не от отсутствия рекламы. Поэтому порядок жёсткий:
**(1) клин-ниша → (2) один город → (3) ликвидность в нише → (4) расширение ниш → (5) расширение городов → (6) платный трафик.**
Платная реклама до ликвидности = сжигание денег на пользователей, которые увидят пустые полки и не вернутся.

### Клин (рекомендация)
1. **Основной клин: подержанная электроника + детские товары** — высокочастотные, компактные (легко встретиться/переслать), самые скамо-ёмкие категории → trust-позиционирование бьёт точно в боль.
2. **Витринный клин: авто** — уже есть уникальный контент (KB поколений: pros/cons/inspection tips, Car-Pass-бейдж). Авто-карточки = SEO-магнит и «wow» продукта, даже если сделки редкие.
3. **Город: Брюссель** — крупнейшая концентрация экспатов (EN/RU/FR аудитория — то, чего у 2dehands нет), затем Антверпен (NL), Гент (NL, студенты).

**Правило контента (жёсткое, из DSA Art.30-дизайна):** никогда не заявлять trust-обещание, которое данные не подтверждают. «Verified» = проверенные email+phone/KBO — формулировки строго фактические.

## 3. Каналы по фазам

### Фаза 0 — Pre-launch (2–4 недели)
- **SEO-фундамент добить** (§7 MASTER_TODO — сейчас отложен, но JSON-LD F12 уже подключён, hreflang есть): sitemap по категориям/городам, OG-картинки. Это единственный канал с компаундом — включать первым.
- Лендинговые страницы категорий `/c/*` (есть) + 10–15 контент-страниц: «Как не попасться на скам при продаже в Бельгии», «Чек-лист покупки б/у авто (по данным KB)», «Car-Pass: что это». Анти-скам-контент = позиционирование + SEO-трафик.
- Сид supply: 20–30 реальных бизнес-продавцов (комиссионки, ремонт телефонов, автодилеры) — онбордить руками, бесплатный Pro на 6 мес. KBO-проверка уже автоматическая.

### Фаза 1 — Soft-launch, Брюссель (месяцы 1–3, бюджет ~€0–500/мес)
- **Community-first:** FB Marketplace-группы, Telegram диаспоры, Reddit — не спам, а полезные посты (анти-скам-гайды) + «мы построили площадку, где это невозможно».
- Студенческие кампусы (ULB/VUB) — сезонная мебель/техника при заездах-выездах.
- Реферальная механика (feature 55, P2 → поднять приоритет): пригласи 3 — получи буст.
- PR-питч локальным tech/consumer-медиа (RTBF, VRT, Brussels Times, l'Echo/De Tijd): «бельгийский стартап против скам-волны» + фактура (данные о фишинге в чатах, которые ловит скраббер).

### Фаза 2 — Ликвидность и расширение (месяцы 4–9, бюджет €2–5k/мес тестово)
- Платный тест: Meta Ads (гео Брюссель, интересы second-hand/parenting/expat) — KPI: ≤€5/регистрация с объявлением; Google Search по «tweedehands + категория» — только после того, как выдача непустая.
- Партнёрства: экспат-организации, kringloopwinkels (circular economy — feature 59 бейдж), velo/детские secondhand-ярмарки.
- Email/push-ретеншен: saved-search алерты (34) + PWA push (18) — это retention-каналы, их надо доделать до платного трафика (см. roadmap).
- Антверпен + Гент: повторить community-плейбук.

### Фаза 3 — Масштаб (месяцы 10+, после Фазы B продукта)
- Запуск safe-deal = второй PR-повод национального уровня.
- TV/radio/OOH — только при подтверждённом LTV/CAC.
- Вертикальные кампании: авто (KB-контент), недвижимость (EPC-бейджи).

## 4. Воронка и целевые метрики

```
Визит → Регистрация → Первое объявление/первый поиск → Контакт → Возврат (D30)
```
| Метрика | Цель Фазы 1 | Цель Фазы 2 |
|---|---|---|
| Объявлений активных | 1 000 | 10 000 |
| WAU | 300 | 5 000 |
| Контакты (диалог ≥3 сообщ.)/нед | 50 | 1 000 |
| D30 retention продавцов | >15% | >25% |
| Органика в трафике | >60% | >50% |
| CAC платный | — | ≤€5/рег. с объявлением |

Аналитика уже в коде: `analytics_events` (F6) + SEARCH_PERFORMED и пр. — дашборд поверх Supabase = первая задача маркетинга.

## 5. Анти-риски GTM

- **Пустые полки:** не открывать категорию в UI-навигации города, пока в ней <30 объявлений (curated launch); zero-result уже конвертирует в SaveSearch.
- **Скам-инцидент в первые месяцы** = смертельно для trust-бренда → ручная модерация первых N тысяч объявлений (admin-панель есть), Turnstile активировать ДО платного трафика.
- **Диаспора-имидж-ловушка:** RU-локаль — тихое преимущество для сообщества, но публичное позиционирование — «5 языков / speaks your language», не моноэтническое.
