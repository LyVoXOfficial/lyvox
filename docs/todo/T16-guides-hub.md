# T16 — Контент-хаб /guides: анти-скам гайды

**Модель:** 🟢 каркас дешёвая (medium) · 🟡 тексты средняя. Каркас механический, тексты требуют аккуратности.
**Ветка:** `feat/guides-hub` · **Приоритет:** P1 (SEO-канал GTM Фазы 1) · **Оценка:** 1 день.

## Зачем
Generic-гайды под скам- и how-to-запросы (nl/fr) — SEO-канал, перехватывающий спрос, где монополист ранжироваться не хочет. Каждый гайд заканчивается ссылкой на категорию + trust-механизмы.

## Образец
`apps/web/src/app/sell/page.tsx` — уже готовая индексируемая SSR-страница с локализованными meta, canonical (`absoluteUrl`), FAQPage JSON-LD, нативными `<details>`. Копируй её паттерн.

## Скоуп
1. **Список гайдов** (10-15) — data-driven, не по файлу на страницу: массив `{ slug, category_link, icon }` в `apps/web/src/lib/guides/index.ts`, тексты в i18n (`guides.<slug>.title/intro/sections[]`) на 5 локалей (приоритет качества nl+fr, en/de/ru — можно короче, но БЕЗ пустых ключей — guard-тест). Темы: «hoe herken je oplichting bij tweedehands kopen», «Car-Pass controleren», «IMEI/чек при покупке электроники», «безопасная встреча для сделки», «EPC при аренде» и т.п.
2. **Роуты**: `apps/web/src/app/guides/page.tsx` (индекс-хаб, список карточек) + `apps/web/src/app/guides/[slug]/page.tsx` (страница гайда: H1 + разделы + Article JSON-LD + ссылка на релевантную категорию `/c/*` + блок trust-механизмов). Оба SSR, `revalidate = 3600`, canonical + локализованные meta.
3. **sitemap**: добавить `/guides` и `/guides/<slug>` в `apps/web/src/app/sitemap.ts` (STATIC_ROUTES или отдельный блок).
4. **Перелинковка**: с релевантных `/c/*` категорий — ссылка на профильный гайд (опционально, если легко); из футера — ссылка на `/guides`.

## Проверка
- `pnpm test` (i18n-паритет!) + typecheck + lint. Локально `curl /guides` и `/guides/<slug>` — 200, JSON-LD присутствует, canonical корректный.
- merge --no-ff, push, прод-проверка.

## Красные линии (ЖЁСТКО)
- **БЕЗ имён конкурентов** и без «безопаснее, чем X» (CEL VI.17 / Dir.2006/114 — запрет дискредитации/паразитирования на чужом знаке). Гайды generic под тот же интент.
- **F3-табу**: никаких «безопасная оплата/защита покупателя/escrow» — только реальные механизмы (верификация, чат, жалоба, встреча в людном месте).
- Тексты — честный образовательный контент, не маркетинговые обещания. i18n 5 локалей, пустых ключей нет.
