## Изменения

- Удалён неиспользуемый `next-i18next`, который приносил critical/high `i18next-fs-backend`.
- Sentry обновлён до Next 16-compatible `@sentry/nextjs@10.65.0`.
- Supabase обновлён до `2.110.2`; типы update payload теперь используют generated `TablesUpdate` вместо небезопасного `Record<string, unknown>`.
- Stripe обновлён до `22.3.1`, API version закреплена как `2026-06-24.dahlia`.
- Lint-only packages перенесены из production dependencies в devDependencies; устаревшие standalone `@types/i18next` и `@types/react-i18next` удалены.
- Transitive `picomatch` обновлён до исправленной версии в рамках разрешённых dependency ranges.
- Patched overrides закрепляют `postcss@8.5.16`, `@babel/core@7.29.7` и исправленные transitive версии audit/tooling; Vite/Vitest/Turbo/js-yaml подняты до безопасных совместимых линий.
- CI проверяет и production graph, и полный workspace/toolchain graph; JSON schema drift и неполные severity details трактуются fail-closed.
- Node runtime закреплён на поддерживаемом Vercel `24.x`, одинаково для workspace, CI и Production deploy.
- Удалён случайный nested npm project `apps/web/apps/web`, не входивший в workspace/runtime.
