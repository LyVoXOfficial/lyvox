2. Фикс бага 1996 — чузер поколения

Переходный год попадает в два поколения сразу. Система не выбирает молча — показывает выбор с отличиями. Это общий приём для всех «диапазонных» справочников.

⚠ Уточните поколение
BMW 5 Series 1996 выпускался в двух поколениях — выберите верное:
E34 1988–1996E39 1995–2003
E34 1988–1996
Прямоугульные фары, «акулий нос». Последний год выпуска.
E39 1995–2003
Овальные фары, новый кузов. Первый год выпуска. похоже на ваши фото
Выбранное поколение сохраняется в объявлении явно (generation_id) — KB больше не «уплывёт».
Логика
  • Кандидаты: start ≤ year ≤ end (или end = NULL). 1 → авто-выбор + «изменить»; >1 → этот чузер; 0 → ближайшие/«неизвестно».
  • Хранить generation_id явно, не ре-деривить из года.
  • Тот же приём: device_models (год релиза), property (год постройки), fashion (коллекция).

3. Мода / личные вещи — раскладка секциями

Полей мало → вкладки избыточны. Один скролл со сворачиваемыми секциями; на мобайле — аккордеон. KB лёгкая: конвертер размеров и (для премиум-брендов) «как проверить подлинность».
