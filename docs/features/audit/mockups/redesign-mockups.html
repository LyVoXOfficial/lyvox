LyVoX — Макеты редизайна

Дизайн · раскладка · функционал. Все ключевые страницы в desktop + mobile, в фирменной палитре LyVoX (teal/mint, Inter, скруглённые карточки). Каждый блок учитывает выводы аудита (см. docs/features/audit/). Цветные пометки: FIX правка по аудиту, NEW новый функционал. Это статичный прототип «как должно выглядеть», не рабочий код.
