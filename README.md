# LyVoX

Welcome to the LyVoX monorepo.

## Structure

- `apps/web`: The main Next.js web application.
- `packages/ui`: Shared UI components.
- `packages/config`: Shared configuration (ESLint, TypeScript).
- `packages/db`: Database schema and migrations.

## Development

1.  `pnpm install`
2.  `pnpm dev`
